# Djnao Project

```angular2html
git clone https://github.com/DonOutcast/module-4-57.git
```
Для запуска проекта переходим в дерикторию advertisements
```angular2html
python manage.py runserver
```

Для запуска Django shell
```commandline
python manage.py shell
```
